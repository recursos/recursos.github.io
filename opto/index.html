Sobre OPTO Sic Notícias

Baixar e guardar localmente vídeos servidos pela OPTO Sic Notícias.

Objectivo: fazer download de vídeos de OPTO Sic
Descrevo a seguir uma tentativa que falhou.
Sem conhecimentos suficientes da tecnologia usada na OPTO, tentei usar uma aplicação (programa) comercial, que tem resultado em sites semelhantes.
Trata-se do IDM (Internet Download Manager), um programa pago, com 30 dias de "free trial".

Mas quem sabe o que vem agarrado ao programa?
Desconfiar sempre destes "free trial"!
Pior, no primeiro uso, o sw esconde no disco rígido informação que dali para a frente impossibilita nova instalação. O que sugere a seguinte precaução, que arruma estes dois problemas:

Correr o programa no Windows Sandbox


A primeira coisa é activar o Windows Sandbox (no Windows 10).
O meu PC é um AMD, talvez as coisas sejam diferentes para INTEL.
A descrição está abreviada, procurar na www informação detalhada e actualizada.
Dando o comando:
cmd\systeminfo.exe:
... Hyper-V Requirements: VM Monitor Mode Extensions: Yes
Virtualization Enabled In Firmware: No
Second Level Address Translation: Yes
Data Execution Prevention Available: Yes

tenho de no BIOS AMD enablar a Virtualizatioon in firmware... será possível?
Foi o que fiz, DEL - OC - CPU Features VBM e ENABLED
Agora task manager\performance diz que está "Virtualization enabled"
Depois seguir:
https://www.windowscentral.com/how-use-windows-sandbox-windows-10-may-2019-update
Control Panel, Windows Features ON/OFF, turn on Sandbox
Seguido de um Reboot
Depois destes passos já me aparece no Start\Windows Sandbox!
